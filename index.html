From Pixels to Precision

Developed a neural network that segments surgical tools from minimally invasive surgical videos, improving surgical precision and safety.

Adversarial Robustness Research

Developed a novel ventral visual neural network, based on CORnet-S architecture with denoising and gated recurrent blocks, achieving robustness of 97.87% under evaluated model threats.

Gesture Controlled Robotic Arm

Developed a robotic arm controlled by hand gestures using computer vision and Arduino, demonstrating real-time human-robot interaction.

Pose Estimation Model

Implemented a Keypoint R-CNN model with OpenCV for real-time pose estimation in security footage and surveillance applications.

Zephyr AI Assistant

Built a keyboard-activated AI assistant that seamlessly integrates into your workflow with a single hotkey press. Leverages Groq's Llama 3.1 8B model to understand natural language commands and instantly control Spotify playback, check weather conditions, aggregate news feeds, manage calendar events, and track project tasks.

DermAI Medical Assistant

Built a DenseNet121 model trained on 19,000+ DermNet images with integrated RAG chatbot for dermatological triage assessment.

Adaptive Learning Robot

A dual-system robot inspired by Daniel Kahneman's System 1 / System 2 model that learns new skills instantly from natural language, without pre-training or rigid scripts. Built with Cohere LLMs, real-time search, and 2 custom 3-DOF robotic arms, the system transforms messy instructions into smooth, executable motions.
