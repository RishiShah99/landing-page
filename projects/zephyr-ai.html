Overview

Zephyr is a personal AI assistant that combines the speed of a minimal popup interface with the power of a full-featured dashboard. Designed for seamless integration into your daily workflow with voice commands, natural language processing, and intelligent memory.

Built to be your always-accessible companion that remembers, assists, and adapts to your needs. Press Ctrl+Alt+Z for instant access, or launch the full dashboard for advanced features and visual management.

The dual interface system provides flexibility: quick commands through the popup (Tiny Zephyr) for speed, and a comprehensive React/Electron dashboard for deep interaction. Seamless transitions between both interfaces with one-click expansion and system tray integration for background operation.

Core Features

Dual Interface System

  • Tiny Zephyr: Instant access popup (Ctrl+Alt+Z) for quick commands and queries
  • Dashboard: Full React/Electron interface for advanced features and visual management
  • Seamless Transition: One-click expansion between interfaces
  • System Tray Integration: Background operation with quick access

Intelligent Interaction

  • Multi-Provider LLM Support: Groq, OpenAI, and Gemini integration
  • Natural Language Understanding: Context-aware command processing
  • Voice Input & Output: Text-to-speech with streaming responses
  • Conversation Memory: Retains context across sessions

Smart Memory System

  • Persistent Fact Storage: SQLite database for fast retrieval
  • Learned Context Retention: Automatic learning from conversations
  • User Preference Tracking: Adapts to your workflow
  • Fast Retrieval: Optimized queries for instant access

Rich Integrations

  • Spotify Control: Play, pause, skip, and track current songs
  • Weather Information: Real-time updates via WeatherAPI
  • File Management: Search and organize files
  • Project Tracking: Idea management and organization
  • Calendar Events: Google Calendar integration
  • Custom Commands: Extensible command system

Architecture

Zephyr uses a three-tier architecture for maximum flexibility:

Tiny Zephyr (Frontend)

  • Minimal Tkinter Interface: Modern dark theme with smooth transitions
  • Global Hotkey Registration: Ctrl+Alt+Z system-wide activation
  • Slide-in Animations: Smooth positioning and auto-hiding
  • Voice Recording: Wake word detection support

API Server (Bridge)

  • Flask REST API: Runs on port 5000
  • Auto-Start: Launches with Tiny Zephyr automatically
  • Command Routing: Bridges UI and backend communication
  • Dashboard Integration: Provides endpoints for full interface

Dashboard (Full Interface)

  • React 18 + Vite: Fast development and hot module replacement
  • Electron: Native desktop experience
  • TailwindCSS: Modern, responsive design system
  • Framer Motion: Smooth animations and transitions
  • Real-Time Sync: Live data synchronization with API server

API Endpoints

The Flask API server provides the following endpoints for dashboard integration:

  • GET /health: Health check and server status
  • POST /command: Execute Zephyr command with natural language
  • GET /memory: Retrieve stored memories and context
  • POST /memory: Add new memory or fact to database
  • DELETE /memory: Remove specific memory entry
  • GET /projects: List all tracked project ideas
  • POST /projects: Create new project entry
  • GET /spotify/status: Current playback information
  • POST /spotify/control: Control playback (play, pause, skip)

Example Usage

curl -X POST http://localhost:5000/command \
  -H "Content-Type: application/json" \
  -d '{"command": "what is the weather"}'

Project Structure

  • app.py: Main entry point for Zephyr
  • ui.py: Tiny Zephyr interface implementation
  • assistant.py: Command handler and routing logic
  • groq_nlp.py: AI/NLP processing with multi-provider support
  • api_server.py: Flask REST API bridge
  • memory_db.py: SQLite database layer for persistent storage
  • projects.py: Project idea management system
  • weather.py: Weather integration via WeatherAPI
  • spotify.py: Music control with Spotipy
  • settings.py: Configuration and preferences
  • tray_icon.py: System tray integration
  • dashboard-app/: React dashboard with Electron wrapper

Known Limitations & Roadmap

Current Limitations

  • Windows Only: Global hotkeys require administrator privileges
  • Single Instance: Only one Zephyr instance can run at a time
  • API Keys Required: Core AI features need Groq/OpenAI API access
  • Network Dependent: Weather, music, and AI features require internet

Future Roadmap

  • Multi-Platform Support: macOS and Linux compatibility
  • Cloud Sync: Optional sync for memories and projects
  • Plugin System: Custom integrations and extensions
  • Mobile App: Companion app for on-the-go access
  • Advanced Calendar: Enhanced scheduling and reminders
  • Team Collaboration: Shared workspaces and projects
  • Offline AI: Local model support for privacy

Conclusion

Zephyr represents the evolution of personal AI assistants: combining instant popup access with a comprehensive dashboard interface. The dual-system architecture ensures you always have the right tool for the task, whether it's a quick voice command or deep project management.

Built with modern web technologies (React, Electron) and a robust Python backend, Zephyr demonstrates how to create flexible, extensible AI systems that adapt to user needs. The three-tier architecture (Frontend, API Bridge, Dashboard) provides a clean separation of concerns and enables independent scaling of each component.

Open-source and ready for extension, Zephyr serves as a foundation for building intelligent assistants that integrate seamlessly into daily workflows.
